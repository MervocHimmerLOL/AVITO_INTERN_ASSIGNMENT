# Автоматизация тестирования API 

Проект автоматизации тестирования REST API сервиса объявлений. Включает позитивные и негативные тестовые сценарии с использованием PyTest и Requests.

## Структура проекта
- config.py Конфигурационные параметры
- conftest.py Фикстуры PyTest
- test_api.py Тестовые сценарии

## Установка и настройка

1. Клонируйте репозиторий
2. Установите зависимости:
pip install -r requirements.txt

## Запуск автотестов
Производится командой pytest test_api.py -v
